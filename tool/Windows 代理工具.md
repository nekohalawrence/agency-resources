# agency-tools

## 内核

- [hiddify-core](https://github.com/hiddify/hiddify-core)
	- 介绍： Hiddify 项目官方核心组件，提供高性能代理内核，支持多种协议与配置格式，常用于 Hiddify 系列客户端的后端实现。
	- 使用教程： https://github.com/hiddify/hiddify-core
	- 标签： #代理内核 #Hiddify #多协议支持 #高性能 #开源核心
- [mihomo](https://github.com/MetaCubeX/mihomo)
	- 介绍： MetaCubeX 维护的 Mihomo 内核（Clash Meta 内核的社区延续版），兼容 Clash 配置，支持更多协议与高级分流规则，是当前主流的 Clash 替代内核。
	- 使用教程： https://wiki.metacubex.one/
	- 标签： #Mihomo #ClashMeta #MetaCubeX #规则分流 #代理内核
- [PuerNya/sing-box](https://github.com/PuerNya/sing-box)
	- 介绍： sing-box 的 PuerNya 分支，通用代理平台内核，支持 Shadowsocks、VMess、Trojan、Hysteria、TUIC 等多种协议，配置简洁且性能优异。
	- 使用教程： https://sing-box.sagernet.org/
	- 标签： #sing-box #PuerNya #通用代理 #多协议 #轻量内核
- [SagerNet/sing-box](https://github.com/SagerNet/sing-box)
	- 介绍： SagerNet 官方维护的 sing-box 内核，通用代理平台，支持丰富协议与出色的跨平台兼容性，是 sing-box 生态的主要分支。
	- 使用教程： https://sing-box.sagernet.org/
	- 标签： #sing-box #SagerNet #通用代理 #协议支持 #跨平台
- [v2ray-core](https://github.com/v2fly/v2ray-core)
	- 介绍： V2Ray 官方核心项目，提供 VMess、VLESS、Shadowsocks 等协议支持，是经典代理工具的基础内核。
	- 使用教程： https://www.v2fly.org/
	- 标签： #V2Ray #v2fly #VMess #VLESS #经典代理内核
- [Xray-core](https://github.com/XTLS/Xray-core)
	- 介绍： XTLS 团队维护的 Xray 内核，是 V2Ray 的增强版，支持更多传输方式（如 XTLS、REALITY）、更高性能与更强的抗检测能力。
	- 使用教程： https://xtls.github.io/
	- 标签： #Xray #XTLS #REALITY #抗检测 #高性能内核
